SkillForgeAI 🚀

SkillForgeAI is an AI-powered freelancing and talent verification platform built to connect skilled freelancers with clients through trust, verification, and smart AI-based matching.

The platform focuses on solving major freelancing problems like fake profiles, payment security, lack of trust, and poor skill validation.

✨ Features
🔐 Secure Authentication
👤 Freelancer Onboarding System
🧠 AI-Based Talent Matching
📊 Trust Score Dashboard
💼 Client & Freelancer Panels
🧪 Skill Verification Tests
💬 Messaging Interface
💰 Wallet & Escrow UI
🛡️ Admin Moderation Panel
📈 GitHub-Style Activity Tracking
